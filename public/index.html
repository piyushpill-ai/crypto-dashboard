BTC/AUD Spread Dashboard

Live bid/ask across Australian exchanges with public APIs.
—
—
Effective price = ask × (1 + taker fee) for order-book venues; retail brokers (CoinSpot/Swyftx) already bake their fee into the quote, so no fee is added. Winner = lowest effective price for the input amount. Samples collected every 4h via GitHub Actions and committed to data/samples.csv in the repo.
